## Project Description

This Java Swing application allows the user to choose one of five pets using radio buttons.

The pets available are:

- Bird
- Cat
- Dog
- Rabbit
- Pig

When a radio button is selected:

- The corresponding pet image is displayed.
- A message box appears showing the selected pet.

---

## Features

- Java Swing GUI
- Five JRadioButtons
- ButtonGroup (only one selection at a time)
- JLabel used to display images
- JOptionPane message dialog
- Event handling using ActionPerformed

---

## Technologies Used

- Java
- NetBeans IDE
- Java Swing

---

## How to Run

1. Open the project in NetBeans.
2. Build the project.
3. Run the project.
4. Select any pet.
5. The image and a message box will appear.
